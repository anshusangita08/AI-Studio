# Milestones

Version: 1.0

---

# Purpose

This directory contains the implementation history of AI Studio.

Each milestone documents a single stage of development, including its objectives, implementation, testing, and completion notes.

Milestones are historical records and should not be rewritten after completion except to correct factual errors.

---

# Usage

When beginning a new milestone:

1. Copy `TEMPLATE.md`.
2. Rename it using the milestone number and title.
3. Complete the document during development.
4. Mark it complete when finished.
5. Update `docs/PROGRESS.md` to reference the new current milestone.

---

# Naming Convention

```text
001-foundation.md
002-project-management.md
003-story-planner.md
...
```

Use a three-digit numeric prefix to preserve chronological order.
