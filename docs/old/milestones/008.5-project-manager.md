# Milestone 008.5: Project Manager (CRUD)

## Overview
Project creation, listing, opening, deletion, renaming, and slug redirection are fully implemented with comprehensive validation.

## Features
- **Create project** – Create a new project directory on disk and register it in the system.
- **List projects** – Retrieve all existing projects.
- **Open project** – Load a project by slug.
- **Delete project** – Remove a project from disk with confirmation handling.
- **Rename project** – Rename both the internal record and the underlying directory, updating slugs accordingly.
- **Slug redirection** – After renaming, redirect to the new slug so URLs remain valid.
- **Reserved name validation** – Prevent creation of projects using reserved names.
- **Duplicate name validation** – Ensure project names are unique.
- **Unit tests** – 30 passing tests covering all CRUD operations.
